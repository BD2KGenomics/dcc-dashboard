You can use the manifest with the Redwood Client to perform a bulk download.

Setting up the Redwood Client

To set up the Redwood Client, you can download it using docker. To download the client, use:
docker pull benjaminran/redwood-client

Getting the token

Contact the admin to get an access token if you don't have one.
